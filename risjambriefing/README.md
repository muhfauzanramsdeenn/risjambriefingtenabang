Aplikasi ini untuk Absensi Rapat Digital Menggunakan Signature Digital
bagi temen temen yang mau mengambil projectnya silahkan dipakai untuk keperluan absensi dan pengembangan aplikasi
Aplikasi ini adalah Aplikasi Absensi Rapat Digital Remaja Masjid Jami Al Makmur (RISJAM) Tanah Abang 
ada fitur tambah rapat, daftar rapat, daftar absensi, daftar kehadiran dan Notulensi
Aplikasi ini diharapkan dapat mempermudah absensi tanpa harus mencatat sehingga datanya sudah bisa dikonversikan ke Microsoft Excel
Silahkan digunakan dengan mendownload zip terlebih dahulu 
Signature digital ini menggunakan Canvas agar bisa tanda tangan di laptop dan mobile (HP)
Setelah itu harus import databasenya di phpmyadmin agar bisa login dan logout
